## рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃЏрЃћрЃбрЃарЃўрЃЎрЃљ: MiFID (Memorization-informed Fr├Еchet Inception Distance)

рЃАрЃБрЃарЃљрЃЌрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃ«рЃарЃўрЃЊрЃљрЃю рЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ 3 рЃърЃўрЃЦрЃАрЃћрЃџрЃў reflection padding-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ. рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЎрЃўрЃЊрЃћрЃћрЃЉрЃў рЃЌрЃўрЃЌрЃЦрЃЮрЃА рЃАрЃљрЃЎрЃБрЃЌрЃљрЃа рЃЌрЃљрЃЋрЃА рЃљрЃўрЃарЃћрЃЎрЃџрЃљрЃЋрЃА рЃЊрЃљ рЃљрЃАрЃћ рЃњрЃарЃФрЃћрЃџрЃЊрЃћрЃЉрЃљ.

Instance Normalization рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃЎрЃЮрЃюрЃбрЃарЃљрЃАрЃбрЃўрЃА рЃЊрЃљ рЃњрЃљрЃюрЃљрЃЌрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃћрЃЉрЃА рЃЊрЃљ рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃЦрЃАрЃћрЃџрЃА рЃАрЃбрЃўрЃџрЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃљрЃерЃў.

4. ReLU Рђћ рЃљрЃарЃљрЃгрЃарЃцрЃўрЃЋрЃЮрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ

рЃЉрЃЮрЃџрЃЮ рЃћрЃбрЃљрЃърЃќрЃћ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ ReLU рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЊрЃћрЃџрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃљрЃарЃљрЃгрЃарЃцрЃўрЃЋрЃЮрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљрЃА. рЃћрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃбрЃўрЃџрЃўрЃА рЃбрЃарЃљрЃюрЃАрЃцрЃћрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃў, рЃ«рЃљрЃќрЃЮрЃЋрЃљрЃюрЃў рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљ.
ReLU рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ:
рЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃю feature-рЃћрЃЉрЃА
рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃљрЃарЃљрЃАрЃљрЃГрЃўрЃарЃЮ рЃАрЃўрЃњрЃюрЃљрЃџрЃА
рЃљрЃЏрЃќрЃљрЃЊрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃА рЃерЃћрЃЏрЃЊрЃћрЃњрЃў, рЃБрЃцрЃарЃЮ рЃдрЃарЃЏрЃљ рЃцрЃћрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА


## Downsampling Рђћ РђърЃЎрЃЮрЃЏрЃърЃарЃћрЃАрЃўрЃљ, рЃарЃЮрЃЏ рЃАрЃБрЃарЃљрЃЌрЃў рЃБрЃЎрЃћрЃЌ рЃњрЃљрЃЋрЃўрЃњрЃЮрЃЌРђю

Downsampling рЃљрЃарЃўрЃА рЃўрЃА рЃћрЃбрЃљрЃърЃў, рЃАрЃљрЃЊрЃљрЃф Generator рЃњрЃљрЃюрЃќрЃарЃљрЃ« рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃАрЃўрЃЋрЃарЃфрЃўрЃЌ рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃљрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃърЃљрЃарЃљрЃџрЃћрЃџрЃБрЃарЃљрЃЊ рЃќрЃарЃЊрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃдрЃарЃЏрЃћрЃА. 
рЃћрЃА рЃћрЃарЃЌрЃў рЃерЃћрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃърЃљрЃарЃљрЃЊрЃЮрЃЦрЃАрЃБрЃџрЃўрЃљ: рЃарЃљрЃбрЃЮрЃЏ рЃЋрЃљрЃърЃљрЃбрЃљрЃарЃљрЃЋрЃћрЃЉрЃЌ рЃАрЃБрЃарЃљрЃЌрЃА, рЃЌрЃБ рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮрЃЊ рЃўрЃАрЃћрЃЋ рЃЏрЃљрЃдрЃљрЃџрЃў рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃњрЃљрЃЏрЃЮрЃАрЃљрЃ«рЃБрЃџрЃћрЃЉрЃљ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ?

рЃърЃљрЃАрЃБрЃ«рЃў рЃљрЃарЃўрЃА рЃўрЃЏрЃљрЃерЃў, рЃарЃЮрЃњрЃЮрЃа рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА рЃюрЃћрЃўрЃарЃЮрЃюрЃБрЃџрЃў рЃЦрЃАрЃћрЃџрЃў:
# рЃърЃўрЃарЃЋрЃћрЃџрЃў downsampling
nn.Conv2d(64, 128, kernel_size=3, stride=2, padding=1),
nn.InstanceNorm2d(128),
nn.ReLU(inplace=True)

# рЃЏрЃћрЃЮрЃарЃћ downsampling
nn.Conv2d(128, 256, kernel_size=3, stride=2, padding=1),
nn.InstanceNorm2d(256),
nn.ReLU(inplace=True)

рЃарЃљ рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃерЃў?

рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃќрЃЮрЃЏрЃљ:
128├Ќ128 Рєњ 64├Ќ64 Рєњ 32├Ќ32

рЃљрЃарЃ«рЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ:
64 Рєњ 128 Рєњ 256

рЃљрЃюрЃБ: рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў рЃўрЃЎрЃБрЃЏрЃерЃћрЃЉрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА РђърЃАрЃўрЃдрЃарЃЏрЃћРђю рЃўрЃќрЃарЃЊрЃћрЃЉрЃљ. рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЦрЃАрЃћрЃџрЃў рЃљрЃдрЃљрЃа рЃцрЃўрЃЦрЃарЃЮрЃЉрЃА рЃфрЃљрЃџрЃЎрЃћрЃБрЃџ рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃќрЃћ Рђћ рЃўрЃА рЃўрЃгрЃДрЃћрЃЉрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃюрЃфрЃћрЃцрЃфрЃўрЃћрЃЉрЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃљрЃА.

рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА рЃћрЃА рЃАрЃљрЃГрЃўрЃарЃЮ?
1. рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃЌрЃў рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃЮрЃЉрЃљ (Computational Efficiency)

рЃЎрЃЮрЃюрЃЋрЃЮрЃџрЃБрЃфрЃўрЃБрЃарЃў рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃФрЃЋрЃўрЃарЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЌрЃЋрЃџрЃўрЃЌрЃў рЃЌрЃЋрЃљрЃџрЃАрЃљрЃќрЃарЃўрЃАрЃўрЃЌ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ: 128├Ќ128 рЃАрЃБрЃарЃљрЃЌрЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ РЅѕ 16-рЃ»рЃћрЃа рЃФрЃЋрЃўрЃарЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ 32├Ќ32 рЃАрЃБрЃарЃљрЃЌрЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ

Downsampling рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА Generator-рЃА: рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃА, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃА рЃЏрЃћрЃбрЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃћрЃџрЃў (features), рЃљрЃа рЃњрЃљрЃЊрЃљрЃўрЃбрЃЋрЃўрЃарЃЌрЃЮрЃА рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљ

рЃћрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ GAN-рЃћрЃЉрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЦрЃАрЃћрЃџрЃў рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА.

рЃўрЃюрЃбрЃБрЃўрЃфрЃўрЃБрЃарЃў рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃљ: рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЏрЃЮрЃюрЃћрЃА рЃюрЃљрЃ«рЃљрЃбрЃА рЃБрЃДрЃБрЃарЃћрЃЉрЃЌ.

рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃ«рЃџрЃЮрЃЊрЃљрЃю рЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃфрЃљрЃџрЃЎрЃћрЃБрЃџ brush stroke-рЃћрЃЉрЃА, рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃљрЃА, рЃАрЃљрЃдрЃћрЃЉрЃљрЃЋрЃўрЃА рЃАрЃўрЃАрЃЦрЃћрЃА
Рєњ рЃћрЃА рЃљрЃарЃўрЃА high resolution, low-level features

рЃерЃЮрЃарЃўрЃЊрЃљрЃю рЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃЏрЃЌрЃћрЃџ рЃАрЃфрЃћрЃюрЃљрЃА, рЃцрЃћрЃарЃЌрЃљ рЃ░рЃљрЃарЃЏрЃЮрЃюрЃўрЃљрЃА, рЃАрЃўрЃюрЃљрЃЌрЃџрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃА
Рєњ рЃћрЃА рЃљрЃарЃўрЃА low resolution, high-level features

Generator-рЃА рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃЮрЃарЃўрЃЋрЃћ:

рЃЦрЃЋрЃћрЃЊрЃљ рЃЊрЃЮрЃюрЃћрЃќрЃћ Рђћ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљ
рЃќрЃћрЃЊрЃљ рЃЊрЃЮрЃюрЃћрЃќрЃћ Рђћ рЃАрЃбрЃўрЃџрЃўрЃАрЃљ рЃЊрЃљ рЃљрЃбрЃЏрЃЮрЃАрЃцрЃћрЃарЃЮрЃА рЃАрЃгрЃљрЃЋрЃџрЃљ

Downsampling рЃќрЃБрЃАрЃбрЃљрЃЊ рЃљрЃЏ рЃЏрЃћрЃЮрЃарЃћ рЃ«рЃћрЃЊрЃЋрЃљрЃА рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА.


### Residual Blocks Рђћ РђърЃњрЃљрЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃљ рЃЊрЃљ рЃарЃћрЃцрЃџрЃћрЃЦрЃАрЃўрЃљРђю

Downsampling-рЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ Generator рЃБрЃЎрЃЋрЃћ рЃљрЃдрЃљрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃќрЃћ. рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ рЃАрЃБрЃарЃљрЃЌрЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф 256 рЃљрЃарЃ«рЃўрЃљрЃюрЃў, 32├Ќ32 рЃќрЃЮрЃЏрЃўрЃА feature map. 
рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЦрЃАрЃћрЃџрЃА рЃБрЃЎрЃЋрЃћ рЃљрЃЦрЃЋрЃА рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџрЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ рЃАрЃБрЃарЃљрЃЌрЃќрЃћ Рђћ рЃўрЃфрЃўрЃА рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃфрЃљ, рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃЏрЃўрЃгрЃљ, рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃгрЃДрЃљрЃџрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃ»рЃћрЃа рЃљрЃа рЃўрЃфрЃўрЃА рЃарЃЮрЃњрЃЮрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃБрЃарЃћрЃЉрЃЮрЃЊрЃћрЃА рЃћрЃА рЃЏрЃЮрЃюрЃћрЃА рЃАрЃбрЃўрЃџрЃерЃў.

рЃљрЃЦ рЃўрЃгрЃДрЃћрЃЉрЃљ CycleGAN-рЃўрЃА Generator-рЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃюрЃљрЃгрЃўрЃџрЃў Рђћ Residual Blocks.

рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃћрЃю Residual Blocks?

Residual Blocks рЃљрЃа рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃћрЃю рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃЊрЃљрЃ«рЃљрЃбрЃЋрЃљрЃА. рЃЏрЃљрЃЌрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљрЃљ рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃљ, рЃўрЃАрЃћ, рЃарЃЮрЃЏ:
рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў (рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў, рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў) рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃЊрЃћрЃА, рЃАрЃбрЃўрЃџрЃў (рЃцрЃћрЃарЃћрЃЉрЃў, рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃљ, brush strokes) рЃЌрЃљрЃюрЃЊрЃљрЃЌрЃљрЃю рЃЊрЃљрЃћрЃЏрЃљрЃбрЃЮрЃА, рЃћрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃ░рЃњрЃљрЃЋрЃА рЃўрЃЏрЃљрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЏрЃ«рЃљрЃбрЃЋрЃљрЃарЃў рЃарЃћрЃљрЃџрЃЮрЃЉрЃљрЃерЃў:
рЃ»рЃћрЃа рЃ«рЃљрЃбрЃљрЃЋрЃА рЃќрЃЮрЃњрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃћрЃЉрЃА, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЋрЃАрЃћрЃЉрЃА рЃцрЃћрЃарЃћрЃЉрЃўрЃЌ рЃцрЃћрЃарЃА, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃљрЃА, рЃЉрЃЮрЃџрЃЮрЃА рЃЎрЃў рЃърЃљрЃбрЃљрЃарЃљ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃА. Generator-рЃўрЃф рЃљрЃАрЃћрЃЌ рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА.
class ResidualBlock(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.block = nn.Sequential(
            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, 3),
            nn.InstanceNorm2d(channels),
            nn.ReLU(inplace=True),
            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, 3),
            nn.InstanceNorm2d(channels)
        )
    def forward(self, x):
          return x + self.block(x)

Residual Connection Рђћ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃўрЃЊрЃћрЃљ

Residual Block-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃљрЃџрЃў рЃљрЃАрЃћ рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ:

Output = ­ЮЉЦ + ­Юљ╣(­ЮЉЦ), рЃАрЃљрЃЊрЃљрЃф:

x Рђћ рЃерЃћрЃЏрЃљрЃЋрЃљрЃџрЃў feature map (рЃарЃљрЃф рЃБрЃЎрЃЋрЃћ рЃЋрЃўрЃфрЃўрЃЌ рЃАрЃБрЃарЃљрЃЌрЃќрЃћ)
F(x) Рђћ рЃўрЃА рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ, рЃарЃљрЃАрЃљрЃф block рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА
рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЦрЃАрЃћрЃџрЃў рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА рЃАрЃ«рЃЋрЃљрЃЮрЃЉрЃљрЃА, рЃЊрЃљ рЃљрЃарЃљ рЃАрЃарЃБрЃџ рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА.

рЃЌрЃБ F(x) = 0, рЃЏрЃљрЃерЃўрЃю: Output = Input -> рЃАрЃБрЃарЃљрЃЌрЃў рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃарЃЕрЃћрЃЉрЃљ

## рЃарЃљрЃбрЃЮрЃЏ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Residual рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ рЃљрЃАрЃћ рЃЎрЃљрЃарЃњрЃљрЃЊ?
1№ИЈ. Gradient Flow Рђћ рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃў рЃАрЃгрЃљрЃЋрЃџрЃљ
рЃдрЃарЃЏрЃљ рЃюрЃћрЃўрЃарЃЮрЃюрЃБрЃџ рЃЦрЃАрЃћрЃџрЃћрЃЉрЃерЃў рЃ«рЃерЃўрЃарЃўрЃљ рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃћрЃгрЃЮрЃЊрЃћрЃЉрЃљ vanishing gradient Рђћ рЃБрЃЎрЃљрЃюрЃљ рЃцрЃћрЃюрЃћрЃЉрЃерЃў рЃњрЃарЃљрЃЊрЃўрЃћрЃюрЃбрЃў рЃАрЃБрЃАрЃбрЃЊрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃгрЃљрЃЋрЃџрЃљ рЃцрЃћрЃарЃ«рЃЊрЃћрЃЉрЃљ.

Residual connection рЃЦрЃЏрЃюрЃўрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃњрЃќрЃљрЃА, рЃарЃЮрЃЏрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌрЃљрЃф gradient-рЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ:
рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃњрЃљрЃЊрЃљрЃЋрЃўрЃЊрЃћрЃА рЃдрЃарЃЏрЃљ рЃцрЃћрЃюрЃћрЃЉрЃўрЃЊрЃљрЃю рЃќрЃћрЃЊрЃљрЃърЃўрЃарЃБрЃџ рЃцрЃћрЃюрЃћрЃЉрЃќрЃћ, рЃљрЃа РђърЃЊрЃљрЃўрЃЎрЃљрЃарЃњрЃЮрЃАРђю рЃЏрЃарЃљрЃЋрЃљрЃџ рЃЎрЃЮрЃюрЃЋрЃЮрЃџрЃБрЃфрЃўрЃљрЃерЃў. рЃћрЃА рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ GAN-рЃћрЃЉрЃерЃў, рЃАрЃљрЃЊрЃљрЃф рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃў рЃўрЃАрЃћрЃЊрЃљрЃф рЃљрЃарЃљрЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃўрЃљ.

2. Identity Mapping Рђћ РђърЃЌрЃБ рЃљрЃа рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ, рЃљрЃа рЃерЃћрЃфрЃЋрЃљрЃџрЃЮРђю

рЃАрЃбрЃўрЃџрЃўрЃА рЃбрЃарЃљрЃюрЃАрЃцрЃћрЃарЃерЃў рЃДрЃЋрЃћрЃџрЃљ рЃарЃћрЃњрЃўрЃЮрЃюрЃА рЃћрЃарЃЌрЃюрЃљрЃўрЃарЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃљрЃа рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
рЃфрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЦрЃЏрЃўрЃА рЃБрЃфрЃЋрЃџрЃћрЃџрЃў рЃЊрЃљрЃарЃЕрЃћрЃА, рЃ«рЃЮрЃџрЃЮ рЃ«рЃћрЃћрЃЉрЃў рЃљрЃю рЃгрЃДрЃљрЃџрЃў рЃБрЃцрЃарЃЮ рЃЏрЃћрЃбрЃљрЃЊ рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃЮрЃА

Residual block-рЃА рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃљрЃЊ РђърЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃўрЃбрЃЮрЃАРђю: рЃЊрЃљ рЃБрЃЉрЃарЃљрЃџрЃЮрЃЊ рЃњрЃљрЃљрЃбрЃљрЃарЃЮрЃА input рЃБрЃфрЃЋрЃџрЃћрЃџрЃљрЃЊ. рЃћрЃА рЃўрЃфрЃљрЃЋрЃА рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃА рЃќрЃћрЃЊрЃЏрЃћрЃбрЃў рЃЊрЃћрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃњрЃљрЃю.

3. Refinement Learning Рђћ рЃћрЃбрЃљрЃърЃЮрЃЉрЃарЃўрЃЋрЃў рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃљ

Residual Blocks рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃЏрЃўрЃДрЃЮрЃџрЃћрЃЉрЃўрЃЌ рЃљрЃа рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃћрЃю рЃарЃљрЃЊрЃўрЃЎрЃљрЃџрЃБрЃа рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃА.
рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў block: рЃЮрЃЊрЃюрЃљрЃЋ рЃљрЃАрЃгрЃЮрЃарЃћрЃЉрЃА рЃцрЃћрЃарЃА, рЃЮрЃЊрЃюрЃљрЃЋ рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃљрЃА, рЃЮрЃЊрЃюрЃљрЃЋ рЃфрЃЋрЃџрЃўрЃА рЃбрЃЮрЃюрЃА

6 block рЃћрЃарЃЌрЃљрЃЊ рЃЦрЃЏрЃюрЃўрЃА рЃЊрЃљрЃ«рЃЋрЃћрЃгрЃўрЃџ, рЃЉрЃБрЃюрЃћрЃЉрЃарЃўрЃЋ рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃљрЃА, рЃарЃљрЃф рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃарЃћрЃљрЃџрЃўрЃАрЃбрЃБрЃарЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ рЃћрЃарЃЌ рЃюрЃљрЃЉрЃўрЃ»рЃерЃў РђърЃњрЃљрЃЊрЃљрЃ«рЃљрЃбрЃЋрЃљРђю.

CycleGAN-рЃўрЃА рЃЮрЃарЃўрЃњрЃўрЃюрЃљрЃџрЃБрЃа рЃюрЃљрЃерЃарЃЮрЃЏрЃерЃў: 256├Ќ256 рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ 9 residual block, 128├Ќ128 рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА 6 block рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃўрЃљ

рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃљрЃарЃўрЃА 128├Ќ128, рЃљрЃЏрЃўрЃбрЃЮрЃЏ: 6 block рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃА рЃАрЃўрЃЏрЃФрЃџрЃљрЃЋрЃарЃћрЃА, рЃљрЃа рЃљрЃЏрЃФрЃўрЃЏрЃћрЃЉрЃА рЃќрЃћрЃЊрЃЏрЃћрЃбрЃљрЃЊ рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃА, рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА over-stylization-рЃўрЃА рЃарЃўрЃАрЃЎрЃА.

### 4. Upsampling Рђћ РђърЃљрЃдрЃЊрЃњрЃћрЃюрЃљ рЃЊрЃљ рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃљРђю

Downsampling-рЃўрЃА рЃЊрЃљ Residual blocks-рЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃў рЃБрЃЎрЃЋрЃћ РђърЃўрЃфрЃюрЃЮрЃЉрЃАРђю рЃАрЃфрЃћрЃюрЃљрЃА рЃБрЃцрЃарЃЮ рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџ рЃЊрЃЮрЃюрЃћрЃќрЃћ (рЃЎрЃЮрЃЏрЃърЃЮрЃќрЃўрЃфрЃўрЃљ, рЃцрЃћрЃарЃўрЃА рЃњрЃљрЃюрЃгрЃДрЃЮрЃЉрЃљ, рЃАрЃбрЃўрЃџрЃўрЃА рЃќрЃЮрЃњрЃљрЃЊрЃў рЃгрЃћрЃАрЃћрЃЉрЃў),

UpSampling #1: 32├Ќ32 Рєњ 64├Ќ64 рЃЊрЃљ 256 Рєњ 128 рЃљрЃарЃ«рЃў

## Transposed Convolution - "рЃерЃћрЃЉрЃарЃБрЃюрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃюрЃЋрЃЮрЃџрЃБрЃфрЃўрЃљ"
ConvTranspose рЃљрЃарЃўрЃА learnable upsampling. 

Regular Conv: рЃерЃћрЃЏрЃЮрЃЊрЃўрЃА рЃЊрЃўрЃЊрЃў рЃАрЃБрЃарЃљрЃЌрЃў, рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА рЃърЃљрЃбрЃљрЃарЃљ feature map
ConvTranspose: рЃерЃћрЃЏрЃЮрЃЊрЃўрЃА рЃърЃљрЃбрЃљрЃарЃљ feature map, рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА рЃЊрЃўрЃЊрЃў рЃАрЃБрЃарЃљрЃЌрЃў 

### Output Layer Рђћ РђърЃцрЃўрЃюрЃљрЃџрЃБрЃарЃў рЃерЃбрЃарЃўрЃ«рЃўРђю

Output Layer рЃљрЃарЃўрЃА Generator-рЃўрЃА рЃЉрЃЮрЃџрЃЮ рЃћрЃбрЃљрЃърЃў 
рЃљрЃЏ рЃћрЃбрЃљрЃърЃљрЃЏрЃЊрЃћ Generator-рЃЏрЃљ рЃБрЃЎрЃЋрЃћ: рЃњрЃљрЃўрЃљрЃќрЃарЃљ рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ (downsampling), рЃњрЃљрЃЊрЃљрЃљрЃЏрЃБрЃерЃљрЃЋрЃљ рЃЊрЃљ рЃАрЃбрЃўрЃџрЃў рЃерЃћрЃБрЃфрЃЋрЃљрЃџрЃљ рЃЏрЃљрЃА (residual blocks), рЃљрЃдрЃљрЃЊрЃњрЃўрЃюрЃљ рЃАрЃўрЃЋрЃарЃфрЃўрЃЌрЃў рЃќрЃЮрЃЏрЃљ (upsampling).

рЃљрЃ«рЃџрЃљ рЃарЃЕрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ:
рЃАрЃбрЃўрЃџрЃБрЃарЃљрЃЊ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃБрЃџрЃў feature map рЃњрЃљрЃЊрЃљрЃўрЃЦрЃфрЃћрЃА рЃюрЃљрЃЏрЃЊрЃЋрЃўрЃџ RGB рЃАрЃБрЃарЃљрЃЌрЃљрЃЊ.

nn.ReflectionPad2d(3),
nn.Conv2d(64, 3, kernel_size=7),
nn.Tanh()

1. рЃарЃЮрЃњрЃЮрЃарЃф Generator-рЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃерЃў, рЃљрЃЦрЃљрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Reflection Padding.
рЃљрЃЏрЃўрЃА рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ, рЃарЃЮрЃЏ рЃЉрЃЮрЃџрЃЮ рЃЎрЃЮрЃюрЃЋрЃЮрЃџрЃБрЃфрЃўрЃљрЃЏрЃљрЃф рЃљрЃа рЃерЃћрЃЦрЃЏрЃюрЃљрЃА рЃЏрЃЎрЃЋрЃћрЃЌрЃарЃў рЃљрЃю рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃарЃў рЃЎрЃўрЃЊрЃћрЃћрЃЉрЃў.
2. 7├Ќ7 Conv Рђћ feature-рЃћрЃЉрЃўрЃЊрЃљрЃю рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃљрЃЏрЃЊрЃћ

3. Tanh Рђћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃАрЃгрЃЮрЃа рЃЊрЃўрЃљрЃърЃљрЃќрЃЮрЃюрЃерЃў рЃЏрЃЮрЃЦрЃфрЃћрЃЋрЃљ
Output Layer-рЃўрЃА рЃЉрЃЮрЃџрЃЮ рЃюрЃљрЃЉрЃўрЃ»рЃў рЃљрЃарЃўрЃА Tanh рЃљрЃЦрЃбрЃўрЃЋрЃљрЃфрЃўрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ. Tanh рЃАрЃўрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃўрЃљ, рЃљрЃюрЃБ: рЃБрЃљрЃарЃДрЃЮрЃцрЃўрЃЌрЃў рЃЊрЃљ рЃЊрЃљрЃЊрЃћрЃЉрЃўрЃЌрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЌрЃљрЃюрЃљрЃЉрЃарЃљрЃЊ рЃљрЃарЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃў. gradient-рЃћрЃЉрЃў рЃБрЃЎрЃћрЃЌ рЃюрЃљрЃгрЃўрЃџрЃЊрЃћрЃЉрЃљ GAN-рЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃюрЃўрЃюрЃњрЃў рЃБрЃцрЃарЃЮ рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃўрЃљрЃћрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃў рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃљрЃа рЃљрЃарЃўрЃА.
